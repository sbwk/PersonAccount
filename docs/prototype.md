# 个人账号管理系统｜界面原型说明（文字版 V0.1）

## 1. 页面清单
- 解锁页
- 账号列表页
- 账号详情页
- 账号编辑页（新增/编辑复用）
- 设置页（最小集）

## 2. 解锁页
- 输入：主密码
- 操作：解锁
- 状态
  - 主密码未初始化：引导设置主密码 + 强度提示
  - 主密码错误：错误提示 + 失败次数提示（不泄露策略细节）

## 3. 账号列表页
- 顶部：搜索框（关键词）、类型筛选（下拉）
- 主区：账号卡片/列表行
  - 标题 + 类型
  - 登录账号（可部分脱敏）
  - 状态角标：即将到期/已到期（依据 expiresAt 或 renewAt）
- 操作
  - 新增：进入账号编辑页（新增模式）
  - 点击条目：进入详情页

## 4. 账号详情页
- 展示：标题、类型、登录账号、url、注册日期、等级、有效期、缴费信息、备注
- 操作
  - 复制登录账号
  - 复制密码（需要点击后才显示或直接复制）
  - 打开网站（url 存在时可用）
  - 编辑
  - 删除（弹窗二次确认）

## 5. 账号编辑页
- 表单字段：按 requirements.md 的字段顺序组织
- 操作
  - 保存：校验通过后写入
  - 取消：返回上一页
  - 生成密码（P2）：在密码输入框旁提供入口
- 错误处理：必填缺失、URL不合法、日期不合法等

## 6. 设置页（V0.1最小集）
- 锁定：立即回到解锁页
- 数据备份/导出：占位入口（V0.2）
- 外观主题：占位入口（V0.2）
