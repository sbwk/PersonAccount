# 个人账号管理系统｜需求清单（V0.1）

## 1. 目标
- 集中管理个人在各平台的账号信息，支持快速检索、跳转登录与密码使用
- 在可控风险下提升便利性：本地/私有化存储优先，提供合理的安全保护与提醒机制

## 2. 范围
- 覆盖“账号条目”的创建、查看、编辑、删除、查询与提醒
- 不包含：自动登录/自动填充（浏览器扩展能力）、多端云同步（除非后续迭代明确）、团队协作

## 3. 数据字段（账号条目 Account）
### 3.1 必填字段
- 账号名称（title）：用于列表展示与搜索（例：Steam 主账号）
- 账号类型（category）：用于分组与样式（例：游戏/工具/邮箱/金融/社交/学习等）
- 登录账号（login）：用户名/邮箱/手机号（可多值，V0.2支持）
- 密码（password）：密文存储；展示时需二次确认或解锁

### 3.2 可选字段
- 注册日期（registeredAt）：日期
- 账号等级（level）：文本或枚举（例：VIP1/Pro/高级会员）
- 有效期（expiresAt）：日期
- 缴费信息（billing）
  - 是否付费（isPaid）：布尔
  - 付费周期（cycle）：月/季/年/一次性/未知
  - 金额（amount）：数字 + 币种（可选）
  - 下次续费日（renewAt）：日期（可与expiresAt并存）
- 官网/登录地址（url）：点击跳转
- 平台密钥（keys）：同一平台可保存多个 Key（例：AI Token），每个 Key 可配置有效期
- 备注（note）：多行文本
- 标签（tags）：字符串数组（V0.2支持）
- 附件/截图（attachments）：文件引用（V0.3支持）

## 4. 功能需求
### 4.1 账号管理（P1）
- 新增账号：支持录入必填字段；可选字段按需填写
- 编辑账号：字段可修改；保存前做基础校验（必填、URL格式、日期逻辑）
- 删除账号：二次确认；支持软删除（V0.2）或直接删除（V0.1可直接删除）
- 账号列表：按更新时间倒序；显示标题、类型、登录账号摘要、有效期/续费状态提示
- 搜索/筛选：按标题、登录账号、类型筛选（V0.1支持关键词+类型）

### 4.2 系统登录与安全（P1）
- 系统主密码（Master Password）
  - 首次启动需设置主密码，并进行强度校验（长度、复杂度、常见弱口令拦截）
  - 登录后在会话期内解锁数据；支持手动锁定
  - 失败重试限制（次数/间隔）以降低暴力尝试风险

### 4.3 快捷使用（P2）
- 点击跳转：从账号条目打开对应网站/登录页（url）
- 一键复制密码：复制到剪贴板；复制后显示短暂提示（不记录明文日志）
- 一键复制登录账号：与密码一致的复制体验
- 一键复制 Key：在详情页对单个 Key 复制；复制后显示短暂提示

### 4.4 平台Key管理（P2）
- 一个账号条目可维护多个 Key（例如 OpenAI/Claude 的 token）
- Key 字段建议：名称/用途、Key 内容、有效期（expiresAt，可选）、备注（可选）
- Key 操作：新增/编辑/删除/复制，按有效期提示“即将到期/已到期”

### 4.5 密码生成（P2）
- 按强度生成：低/中/高三档
  - 低：字母+数字
  - 中：字母（大小写）+数字
  - 高：字母（大小写）+数字+符号，长度更长
- 生成后可一键填入“密码”字段，并支持再次生成

### 4.6 续费/到期提醒（P2）
- 基于 expiresAt 或 renewAt 提醒
- 提醒规则（默认）：到期前 7 天、3 天、1 天；到期当天
- 提醒方式（V0.1）：应用内提醒/列表高亮；（V0.2：系统通知/邮件由用户选择）

### 4.7 个性化（P3）
- 页面风格：浅色/深色/跟随系统（V0.2起）
- 账号类型自定义：新增/编辑类型、图标/颜色（V0.3起）

## 5. 约束与非功能性需求
- 不在任何日志、报错、埋点中输出明文密码
- 支持数据备份与导出（V0.2）：导出需加密或由用户确认风险
- 关键操作需要可恢复性（V0.2软删除/回收站）

## 6. 优先级（确认版）
- P1：账号信息存储与管理、系统登录主密码强度校验
- P2：快捷跳转、一键复制（含Key）、平台Key管理、自动生成密码、续费提醒
- P3：页面风格、账号类型自定义

## 7. 验收标准（V0.1）
- 能创建/编辑/删除/查看账号条目，字段校验符合预期
- 主密码未登录状态下无法查看任意账号明文信息
- 列表可检索并能从条目跳转到 url
- 复制登录账号/密码在成功与失败时有明确反馈
