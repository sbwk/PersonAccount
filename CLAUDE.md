# PersonAccount

个人账号存储管理系统

## 快速开始

### 安装
```bash
cd python
pip install -r requirements.txt
pip install -e .
personaccount
```

### 构建
```bash
make build
```

### 测试
```bash
make test
```

## 项目结构

- `python/` - Python 后端代码
- `docs/` - 项目文档
- `Makefile` - 构建命令
