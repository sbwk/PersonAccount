# 安全策略

## 报告安全问题

如果您发现 PersonAccount 的安全漏洞，请：

1. **不要**公开披露
2. 发送邮件至 [your-email@example.com]
3. 或在 GitHub 上 privately report a security vulnerability

## 我们如何处理安全问题

- 我们将在 48 小时内确认收到您的报告
- 我们将在 7 天内提供初步评估
- 修复后，我们将发布安全公告

## 支持的版本

我们仅为最新版本提供安全更新。

| 版本 | 支持状态 |
|------|----------|
| 0.1.x | 当前版本 |
